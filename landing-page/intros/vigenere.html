The Vigenère Cipher

"Le Chiffre Indéchiffrable" (The Indecipherable Cipher)

Beyond Simple Substitution

For centuries, simple substitution ciphers were easily cracked using frequency analysis. In the 16th century, Blaise de Vigenère popularized a method that used multiple substitution alphabets to hide the frequency patterns.

How it Works

The Vigenère cipher uses a keyword (e.g., "KING"). The message is encrypted using a series of Caesar ciphers based on the letters of the keyword.

  • The first letter of the message is shifted by the first letter of the key ('K').
  • The second letter is shifted by the second letter of the key ('I').
  • ...and so on, repeating the keyword as needed.

This means the letter 'E' might be encrypted as 'P' in one place and 'X' in another, making frequency analysis much harder.

The Vigenère Square

To perform the encryption manually, cryptographers used a tabular tool called the Vigenère Square (or Tabula Recta). It consists of 26 alphabet rows, each shifted one position to the left.
