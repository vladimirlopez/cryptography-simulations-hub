# Caesar Wheel Simulation

An interactive Caesar Wheel simulation designed for educational purposes (9th grade cryptography).

## Features
- Interactive 3D-like Caesar Wheel (SVG)
- Real-time encoding and decoding
- Challenge mode for students
- Consistency with "The Thinking Experiment" blog aesthetic

## Tech Stack
- HTML/CSS (Vanilla)
- JavaScript (Vanilla)
- Vite (Dev server and build)

## Local Development
```bash
npm install
npm run dev
```

## Deployment
Automatically deployed to GitHub Pages via GitHub Actions.
